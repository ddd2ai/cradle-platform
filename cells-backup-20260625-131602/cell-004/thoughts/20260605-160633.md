    ### Task
    設計 Colony 共用記憶體制

    ### Source

    ### Result Summary
    任務理解  
本任務需設計並實作一套 Colony 級別的共用記憶體制，讓所有細胞能即時同步狀態、協作任務、累積知識，並具備細胞身份驗證與資料權限控管，避免資訊孤島與權限濫用。

執行結果  
- 架構規劃：建議採用中心化記憶體（如 Redis、分散式 DB）或事件流（如 Kafka）作為底層，設計 API 供細胞讀寫、訂閱、比對差異。
- 權限設計：每筆資料標註細胞身份與權限，支援細胞級存取控管。
- 初步 PoC：可先以 Redis 實作 key-value 結構，驗證同步與協作流程。

下一步建議  
1. 明確定義共用記憶的資料結構與 API，並文件化。
2. 實作權限控管與版本追蹤機制，確保資料一致性與安全。
3. 與 DNA/任務系統整合，讓細胞能自動讀寫共用記憶。
4. 持續收集協作痛點，優化記憶體設計與操作體驗。
