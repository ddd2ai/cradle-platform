執行結果

  - POST /payments/refund

- 定義 Payment、Refund Entity 與 DTO

- Service：封裝付款/退款邏輯  
- Controller：接收請求並回傳結果  
- 整合 Springdoc/OpenAPI，自動產生 API 文件  
- 初步設計資料結構，考量敏感資訊加密與權限驗證
