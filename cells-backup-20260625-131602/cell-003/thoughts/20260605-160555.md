    ## Task Experience

    ### Task
    設計協作與知識共享流程

    ### Source
    metabolism

    ### Result Summary
    任務理解  
本任務需設計一套細胞協作與知識共享流程，讓多細胞能協同處理複雜任務，並將過程中產生的發現、最佳實踐與問題同步給整個 colony，避免重工與知識孤島。

執行結果  
- 協作機制：任務支援多細胞協作，協作者可即時更新進度、分工與狀態，並記錄協作歷程。
- 任務協商：建立協商 API，細胞可主動發起協作邀請、協商 ownership 或釋放任務。
- 發現共享：所有任務、討論與 artifacts 統一存放於共享 workspace，並定期廣播重要發現或決策，支援全文檢索與訂閱。
- 知識同步：每次任務結束自動產生 knowledge snapshot，供 colony 內所有細胞查閱與學習。

下一步建議  
1. 擴充任務與協作資料結構，支援多協作者與知識標註。
2. 實作協作邀請、協商與知識廣播 API。
3. 建立 artifacts 與 knowledge snapshot 的存放與檢索規範。
4. 撰寫單元測試，驗證協作與知識同步流程正確性。
