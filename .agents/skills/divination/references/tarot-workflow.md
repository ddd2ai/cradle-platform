# Tarot Workflow（固定順序）

## 0. Input Check

- **⏰ 先取得當前時間**（用於時間建議錨定）
- 所有輸入皆為可選：
  - `spreadType` — 牌陣類型：
    - `single` — 單牌（快速回答，預設）
    - `three-card` — 三牌（過去/現在/未來，標準場景）
    - `love` — 愛情牌陣（感情關係專題）
    - `celtic-cross` — 凱爾特十字（深度多維度解讀）
  - `question` — 占卜問題（可選，建議聚焦單一主題）
  - `allowReversed` — 是否允許逆位（預設 true）
  - `seed` — 隨機種子，固定後可重現
- 牌陣選擇策略：
  - 簡單是非問題 → `single`
  - 帶時間脈絡的問題（過程/發展） → `three-card`
  - 感情相關 → `love`
  - 複雜人生重大問題 → `celtic-cross`
- 抽牌前確認問題：聚焦單一主題（關係/事業/決策）
- 無明確問題時，預設「當前狀態 + 下一步建議」

## 1. 抽牌層

- 調用 `tarot_draw`，取得：
  - `spreadName` → 牌陣名稱
  - `cards[]` → 抽到的牌
  - `seed` → 本次種子（便於重現）
- 如需可重現，設定 `seed` 參數

## 2. 單牌語意層

- 逐張遍歷 `cards[]`，擷取：
  - `position` → 牌位含義（如「過去」、「現在」、「未來」、「阻礙」等）
  - `card.nameChinese` → 中文牌名
  - `card.keywords[]` → 關鍵詞列表
  - `orientation` → 正位（upright）/ 逆位（reversed）
  - `meaning` → 工具提供的牌義
- 逐張解釋：
  - 牌位含義
  - 正逆位含義
  - 該牌對問題的直接指向

## 3. 牌陣關係層（Mandatory）

- 不只解讀單牌，必須說明牌與牌之間的關係：
  - 支持 / 阻礙
  - 過去－現在－未來的遞進關係
  - 內因 / 外因的區分

## 4. 行動建議層（Mandatory）

- 輸出短期行動（3–7 天）
- 輸出中期行動（1–4 週）
- 輸出「不建議做」的行動（避免誤判）

## 5. 風險與邊界層

- 塔羅用於決策輔助，不取代醫療/法律/投資等專業意見
- 高風險問題需增加理性驗證建議

## 6. Quick Template

1. 結論摘要（當前趨勢）
2. 牌陣核心矛盾（支持 vs 阻礙）
3. 時間建議（短期 / 中期）
4. 行動清單（建議做 / 不建議做）
