# Todo API 設計

## 目標
提供一組簡潔、可擴充的 Todo API，支援基本任務管理操作，並預留未來整合 Merlin Platform 的彈性。

## 路由設計

### 1. 取得所有 Todo
- **GET** `/api/todos`
- 回傳所有 Todo 項目列表

### 2. 新增 Todo
- **POST** `/api/todos`
- 請求內容：
  ```json
  {
    "title": "string",
    "description": "string"
  }
  ```
- 回傳新增後的 Todo 物件

### 3. 取得單一 Todo
- **GET** `/api/todos/{id}`
- 回傳指定 id 的 Todo

### 4. 更新 Todo
- **PUT** `/api/todos/{id}`
- 請求內容：
  ```json
  {
    "title": "string",
    "description": "string",
    "status": "pending | in_progress | done"
  }
  ```
- 回傳更新後的 Todo

### 5. 刪除 Todo
- **DELETE** `/api/todos/{id}`
- 回傳刪除結果

## 資料結構

```json
{
  "id": "string",
  "title": "string",
  "description": "string",
  "status": "pending | in_progress | done",
  "created_at": "ISO8601 string",
  "updated_at": "ISO8601 string"
}
```
