### Goal

寫一個單檔可執行 Java 程式,名稱為 BankTransferApp。這個程式要模擬一個迷你銀行轉帳清算系統。請包含 Account、TransferRequest、TransferResult、LedgerEntry、BankService、LedgerService 這些類別或 static nested class。main 方法中要建立三個帳戶,分別是 Alice、Bob、Charlie,設定初始餘額。接著執行三筆轉帳: Alice 轉 120.50 給 Bob, Bob 轉 50.00 給 Charlie, Charlie 嘗試轉 9999.00 給 Alice。每筆成功轉帳都要扣除 1.00 手續費,餘額不足時要拒絕交易且不可扣款。最後輸出每筆交易結果、所有帳戶最終餘額、ledger 明細、成功交易數、失敗交易數、總手續費。程式不可使用外部套件,只能使用 Java 標準函式庫。執行結果必須輸出 transfer accepted, transfer rejected, final balances, ledger entries, total fee 這些資訊。
