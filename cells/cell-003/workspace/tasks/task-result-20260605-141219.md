    ## Task

    設計與實作查詢分頁機制

    ## Task ID

    task-20260605-141210

    ## Result

    任務理解  
本任務需為商品查詢 API 增加分頁功能，API 需支援 page/size 參數，Repository 層實作分頁查詢，回傳分頁資訊，提升查詢效能與可用性。

執行結果  
- API 設計：新增 `page`（頁碼）、`size`（每頁筆數）查詢參數，預設值可為 page=0, size=20。
- Repository 實作：使用 Spring Data JPA 的 `Pageable` 或 MyBatis 的分頁查詢，回傳 Page 物件或自訂分頁結構。
- 回傳格式：包含商品清單、總筆數、總頁數、當前頁碼等分頁資訊。

下一步建議  
1. 前端/呼叫端需支援分頁參數。
2. 撰寫分頁查詢單元測試，驗證正確性與效能。
3. 與快取、索引優化任務協同進行，確保整體查詢體驗。
