# DNA Factors

This file defines the measurable factors used by every DNA trait.

Each Cell will generate its own dna-vector.json based on DNA_DEFINITION.md and DNA_FACTORS.md.

---

## strength

How strongly this DNA trait influences the Cell behavior.

## stability

How resistant this DNA trait is to change.

## plasticity

How easily this DNA trait can be reshaped by new stimulus.


## fitness

How effective this DNA trait is in the current environment.
