# DNA Definition

This file defines the DNA traits available in this Cradle.

Each Cell will create its own DNA files based on this definition.

---

## PERCEPTION

How a Cell senses external stimulus.

## DECISION

How a Cell makes choices.

## DECOMPOSITION

How a Cell breaks large problems into smaller parts.

## LEARNING

How a Cell absorbs experience.

## COLLABORATION

How a Cell cooperates with other Cells.

## CREATION

How a Cell transforms work into artifacts.

## EVOLUTION

How a Cell adapts and changes over time.

## REFLECTION

How a Cell reflects on its own behavior.
